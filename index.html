pretext demo

DOM-free text measurement & layout by @chenglou/pretext

1. Instant Text Measurement prepare + layout

Type text and drag the width slider. prepare() measures once via canvas; layout() recalculates height with pure arithmetic on every resize — no DOM reflows.

2. Manual Line Layout layoutWithLines

Get back individual lines with their text, width, and cursor positions — perfect for rendering text on Canvas or SVG without the DOM.

3. Variable-Width Wrapping layoutNextLine

Use layoutNextLine() to wrap text around obstacles — each line can have a different available width. Here, text wraps around a floating "image".

4. Inline Flow — Mixed Fonts & Chips experimental

The inline flow module handles rich text with different fonts and atomic "chip" elements that never break mid-word, like tag pills in an editor.

5. Performance Comparison DOM vs pretext

Measuring 500 text blocks: DOM uses getBoundingClientRect() (triggers reflow each time), while pretext uses prepare() once then layout() for pure math.
